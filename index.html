Not Connected v3.0 WebUSB

1. Sambungkan Peranti (WebUSB)

Sambungkan telefon ke PC menggunakan kabel. Website ini akan berkomunikasi terus dengan telefon anda tanpa perlu install sebarang perisian (software) di PC.

Langkah Wajib:

  1. Aktifkan Developer Options di telefon.
  2. Hidupkan USB Debugging.
  3. Klik butang di bawah dan pilih telefon anda dari senarai (popup).
Device Connected

*Sistem berfungsi di Chrome, Edge, dan pelayar berasaskan Chromium.

2. Scan & Buang Iklan

Imbas (Scan) telefon anda untuk mencari aplikasi tersembunyi yang menyebabkan iklan pop-up (adware).

Panduan Bantuan

  1. 1

    Gagal Bersambung?

    Jika anda pengguna Windows, anda mungkin perlu install WinUSB driver menggunakan perisian Zadig untuk membolehkan Chrome membaca ADB.

  2. 2

    Server Tempatan

    Pastikan anda membuka laman web ini menggunakan HTTPS atau localhost. WebUSB tidak berfungsi jika dibuka menggunakan 'file://'.

  3. 3

    Kebenaran (Allow)

    Selepas klik Connect, sentiasa lihat skrin telefon anda. Satu popup "Allow USB Debugging?" akan muncul. Tekan Allow.
